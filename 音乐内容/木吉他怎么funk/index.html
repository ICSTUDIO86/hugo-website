什么是 Mixolydian 调式？

你可以把一个 Mixolydian 音阶想成是一个降了七音的大调音阶。例如，C Mixolydian 就是 C 大调音阶，但把 B 降低半音，而这个降七音正是这个音阶的特色音。新获得的这个音阶的全半关系是：全全半全全半全，你会发现如果我们把一个 C 大调音阶里的 G 当作主音然后弹一遍 C 大调音阶里的音，你也会获得相同的全半关系。

Mixolydian 的和声进行

因为调式发生了变化，和声的运作逻辑也会随之不同。我们不能按照传统大小调来配和弦，而是需要基于 Mixolydian 调式的特性来构建和声进行。那 Mixolydian 调式有哪些特性呢？为了方便理解，我接下来会以 C Mixolydian 作为例子，但同样的理论你一样应用到其他 Mixolydian 调式里。 首先第一个特征是他的主和弦，虽然主和弦的三和弦和一般大调一样是一个大三和弦，但是如果把这个和弦上升到七和弦的话那么他就是一个属和弦了，然后属七 sus 也是很常会在 Mixolydian 歌曲里被当作主和弦使用。另外一个会凸显 Mixolydian 色彩的和声进行是它的特色和弦。所谓的特色和弦就是有特色音的和弦，不包括主和弦。 在 C Mixolydian 里我们的特色和弦分别是，三级和弦，Edim 或者 Eø，五级和弦，Gm 或者 Gm7，七级和弦，Bb 或者 Bbmaj7。我们只要不断重复地从一个特色和弦接到主和弦，就可以制造出一个具有 Mixolydian 色彩的和声进行了。就像这样。你也可以用两个特色和弦接主和弦。就像这样。

用 Mixolydian 给旋律重配和声

那怎么给一个旋律重新配上一个 Mixolydian 的和声呢？在今天这个例子里，旋律虽然是 C 大调的，但里面的音正好也属于 G Mixolydian 音阶。所以我们使用 G Mixolydian 的和声听起来就会和这个旋律很般配。在给 Mixolydian，或者其他非大小调的现代中古调式配和声时，你需要格外的强调调式里的主和弦。在今天这个例子里，几乎每两个小节你就可以听到一次主和弦，因为这样会强调 Mixolydian 的听感。 你一般会想要把主和弦放到重音上，例如一个乐句的开头，或者一个乐句的结尾，又或者是单数小节上，因为像这种以四个小节为一个单位的音乐，重拍都会落在单数小节上。而在这些主和弦的前面你就需要放我们刚刚提到的特色和弦来再次强调这个音阶的特色音，以此进一步的凸显 Mixolydian 独有的色彩。如果在 G Mixolydian 里面我不强调主和弦的话，那么这首歌就会听起来像 C 大调。 首先是因为这个旋律本身就是一个 C 大调的旋律，其次是音阶里的 B 会有一个需要解决会 C 的倾向，然后音阶里的三全音也会有一个解决会 C E 的倾向。所以如果你不用什么特别手法来强调主和弦或者特色音的话，那这个 G 会听起来像 C 大调里的属和弦。所以在配这一类和声的时候，强调主和弦是一个非常重要的事情。
