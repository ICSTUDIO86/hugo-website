吉他手需要知道关于五线谱的所有事情

2025年12月24日 ； 阅读时间约 20 分钟

开头

这个频道其实已经不只一次强调过五线谱对吉他手的重要性了。如果要用几句话把它总结出来，那就是：五线谱会逼迫你在指板上不断找音、定位音高，在练习的过程中自然建立你的指板地图。这样一来，不管是即兴、合奏还是编曲，你的反应都会更快，你也会更清楚自己到底在弹什么。不过虽然我们之前一直在讲五线谱的重要性，这个频道好像还没有做过一支真正系统、清楚地讲解“五线谱到底要怎么看”的视频。所以今天这期视频就是要补上这个空白。

五线谱与谱号：最小必要认知

首先，五线谱，顾名思义，是由五条横线组成的谱子。音符可以落在线上面或者两条线之间，这些线与间都是从下往上数的。音出现在越上面，就代表这个音越高，相反，如果越在下面，音高则越低。如果需要的话，你可以往上或者往下再加几条线来表示更高或者更低的音。哦对，如果音头出现在五线谱的上半部分，那么符干就要朝下，如果在下面，那么符干就要朝上。而且符干的长度要刚好是一个八度。 不过要具体知道一个音符在五线谱上对应的是什么音，我们得先认识谱号，就是这个会出现在五线谱开头的这图案。谱号的位置以及图案会直接决定五线谱的这些线与间对应的是哪些音。在五线谱的系统里一共有三种谱号：高音谱号，中音谱号，低音谱号。高音谱号的意思就是任何出现在穿过高音谱号肚子的那条线是 G4，也就是中央 C 往上的那个 G。，一般穿过高音谱号肚子的线是第二条线。 吉他这个乐器基本都用高音谱号，不过会默认整体音高会低一个八度，也就是原本这里是 G4，在吉他上就是 G3，也就是中央 C 往下的那个 G。有些时候在谱号下方还会有一个小数字“8”用来表示整体音域降低了一个八度。由于吉他不使用中音谱号和低音谱号，所以在这里我就简单的概括一下。中音谱号的意思就是任何出现在穿过这个符号的中间的线的音符就是 C4，也就是我们所说的中央 C。 这个谱号会在在管弦乐配器里常见；低音谱号的意思就是任何穿过两个点之间的线是 F3，也就是中央 C 下面的 F。这条线通常是五线谱的第四线。主要是给低音乐器用的。例如低音提琴，贝斯等乐器。因为不同的乐器和人声有不同的音域，所以就会有不同的谱号，这样就能让音符尽量落在五线谱范围内，减少加线，方便阅读。如果所有乐器都用同一个谱号，比如高音谱号，那么像是贝斯，大提琴这样的低音乐器就需要使用大量加线，给乐手增加不必要的麻烦。

节奏

接下来我们来看看节奏在五线谱里是如何表达的，也就是音符的长短。节奏可以说是音乐里最重要的元素之一了。音符的节奏是由它的外形来决定的。在五线谱里，一个音符通常由三部分组成：符头、符干、以及符尾。像这个由一个实心符头加一个符杆形成的音符就叫作四分音符。比四分音符快一倍的音符叫作八分音符，八分音符是在四分音符的基础上再加上一条符尾。如果四分音符的时值相当于一秒的话，那么八分音符的时值就会是 0.5 秒。 比八分音符再快一倍的音符叫作十六分音符。十六分音符有两条符尾，以此类推——之后每当音符的时值缩短一半，就会多出一条符尾。而当多个带符尾的音符连在一起时，它们的符尾会合并。比四分音符慢一倍的音符叫作二分音符，二分音符和四分音符一样没有符尾，不过它的音头是空心的。再慢一倍就是全音符，全音符的符头是空心的，并且没有符干。在五线谱里，休止符也有它自己的一套长相，休止符就是你休息的时长。 全音符的休止符和二分音符的休止符长的非常像，其实分辨的方法很简单，在一个四四拍里，如果这个小节是空的，那么你就可以猜出来他是一个全音符的休止符，如果小节里还有其他音符，那么就是一个二分休止符。这个 上面一个 Z 加下面一个 C 的图形是四分休止符，最后这个小弯钩就是八分休止符，十六分休止符则是有两个弯钩。依次类推，每当音符的时值缩短一半，就会多出一个小弯钩。 然后还有一个特殊的音符叫做附点音符，就是在音符旁点一个点，加了点的音符时值就等于音符原来的时值加上它的一半；例如附点四分 = 四分 + 八分。如果四分音符等于一秒的话，那么附点四分音符就等于 1.50 秒。另外我们还会看到一种非常常见的弧线，也就是连线。连线有两种意义，取决于它连接的音高是否相同：如果连的是两个相同音高的音符，那叫做延音线（Tie）。 你只需要弹第一下，然后这个音的总时值是两个音符的时值加起来；如果连的是两个不同音高的音符，那就是连奏线（Slur）。在吉他上，这通常表示要用左手技巧来完成连奏：往高音连就是击弦（Hammer-on），往低音连就是勾弦（Pull-off），不需要用右手再拨一次弦。不过有时候你也会在连奏线上方看到 H 或 P 的标记，分别用来提示击弦和勾弦。 除此之外，你偶尔也会看到一种跨越好几个音符的长连线，它是用来标记一句音乐的起伏与范围，也就是所谓的乐句连线（Phrase Slur）。在节奏里，我们还会遇到一种特殊的节奏分组叫做连音。连音的意思就是把原本的一拍平均分成 X 份，而 X 就是你在连音上方或者下方会看到的那个数字。例如最常见的就是三连音（Triplet）。如果四分音符是一秒的话，那么三连音的每一个音符差不对会是 0.33 秒。

拍号

在五线谱开头的地方我们还可以看到一个类似分数的东西，但这个不是分数，你也千万不要傻傻的将它进行约分。这个在音乐里叫作拍号。拍号是一首歌的节奏规则。我们都知道，人类就是一个喜欢将事情进行归类的一个物种，音乐当然也不例外。如果没有规律地将节奏进行量化或者排序，那音乐就会变得非常混乱——你不知道从哪里开始数拍，也不知道什么时候应该一起演奏或停下来。 拍号会告诉你两件很重要的事情：1. 上面这个数字会告诉你一个小节里面要数几拍，小节就是一个被这些线给分割出来的区域。2. 下面这个数字会告诉你哪一种音符算一拍 例如我们现在看到的这个 3/4 拍的意思就是在说一个小节有 3 拍，然后四分音符算作一拍。或者 6/8 拍，意思是每小节要数六拍，然后每一拍的长度是八分音符。有些时候，4/4 拍也会用这个半圆形来表示。 会选择用半圆来表示 4/4 拍的原因，是因为在西方宗教的理念里，数字三一直被视为一个神圣、完美、完整的数字，而数字四相对来说就显得“没那么神圣”。所以在早期的音乐记谱系统里，用一个“完整的圆”表示的是三拍子的曲子，而用一个“被切开的圆”——也就是半圆形——来表示四拍。

Tempo

刚刚在讲音符时值的时候，我暂时把四分音符当成一秒来理解，但真实情况下它可以是任何长度。音符之间真正重要的是它们的倍数关系——谁快一倍、谁慢一倍——而不是真实的秒数。 真正决定一个音符具体时长的是 Tempo。也就是在五线谱的最开头，你还会看到一个表示速度的标记。 这个「♩ = 100」的意思就是说，这个曲子的速度是每分钟 100 个四分音符。而四分音符刚好也是这个曲子的一拍，所以意思就是说每分钟都有 100 拍。 我们也可以把它说成 100 BPM（Beats Per Minute）。速度也可以用意大利文来描述。 这些速度标记会给你大概描述一下整体的节奏氛围，让你大概知道这首曲子应该以一个怎样的速度去演奏。

调号

还有一个需要了解的就是调号了，在五线谱上，调号就是通过谱子一开始的这些升降记号来标记的。这个调号其实就直接决定了一首曲子的默认升降规则。如果谱子的开头没有标记任何升降记号，那就代表这首曲子没有默认的升降音，所有的音都按照自然音来演奏。但如果在一开始 F 这里被加入了一个升记号，那么就代表接下来所有的 F 都要演奏成 F#，不管它是五线谱上哪一个 F。 同样的道理，如果 E，B，A 这里被加入了一个降记号，那么就代表接下来的所有 E，B，A 都要被演奏成 Eb，Bb 和 Ab。不过你还是有办法在五线谱内加入一些临时记号，来暂时修改一个音符。不过这些临时记号只能持续一个小节。在曲子到下一个小节的时候，这个音符会自动回到默认状态，也就是调号的设定。 临时记号有升记号，意思就是让一个音升高一个半音，重升记号，就是让一个音升高两个半音，降记号，意思就是让一个音降低一个半音，重降记号就是让一个音降两个半音，最后是还原记号，也就是让这个音回归到没有任何升降记号的状态。

小节线（Barlines）

接下来我们来看看小节线。在五线谱里，我们会看到各种不同类型的“分割线”，它们叫做 小节线（Barlines），用来组织音乐段落，让演奏家知道结构、重复方式以及段落的变化。常见的有：单小节线，就是最普通的一条线，用来分隔每一个小节。然后是双小节线，由两条细线组成，通常表示一个段落结束，或者音乐在结构上要进入一个新的内容了。例如主歌已经结束了，双小节线之后的就是副歌了。 接着是终止线，由一条细线加一条粗线组成，表示音乐到这里就正式结束了。最后就是重复线。在五线谱里，我们还会看到一种带着两个小点的双小节线，这就是所谓的重复线。它的作用是让你把某一段音乐重新演奏一次。被重复的内容其实很好判断：就是被两个带点的小节线所框起来的那一段区域。也就是说当你从开头弹到这里的时候，你需要重新回到开头这个反复记号然后接着往下演奏。 不过当一段音乐需要重复，但它的结尾不同的时候，你就会看到写着“1.”和“2.”的那种小括号，它们分别代表第一结尾和第二结尾。演奏方式也很简单：第一遍你就把第一结尾的内容正常弹完；遇到重复线之后回到前面，再重弹一遍。当你第二次来到同一个位置时，就不要再进入第一结尾了，而是直接跳到第二结尾，继续往下演奏。另外在一些谱子里，你还会看到一个看起来像百分号的符号「%」。这个符号的意思是：这个小节要完全重复上一个小节的内容。
