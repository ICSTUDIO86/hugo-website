吉他上怎么按布鲁斯音阶？练习或者即兴的时候应该注意什么？

2025年1月17日 ； 阅读时间约 9 分钟

什么是布鲁斯音阶？

那到底什么是布鲁斯音阶呢？布鲁斯音阶也被叫做蓝调音阶。这个就是我们常说的布鲁斯音阶，它是一个小调五声音阶加上了一个与主音形成三全音的音，而这个音阶也被我们成为小调布鲁斯音阶。在这个情况我们是拿一个 A 小调五声音阶：A，C，D，E，G，然后在上面加上一个与主音形成三全音的音，也就是 Eb。听起来就会像这样。有小调布鲁斯音阶自然就有大调布鲁斯音阶，这个大调布鲁斯音阶就是一个大调五声音阶加上一个与主音能形成小三度的音。 例如我们有一个 C 大调五声音阶：C，D，E，G，A，然后我们加上一个 Eb，这个音阶听起来就会像这个样子。和大小调的逻辑一样，C 大调布鲁斯音阶和 A 小调布鲁斯音阶用的是同一个音，所以它们也是彼此的关系大小调。这个比起五声音阶多出来的这个音被叫作 Blue Note，就是因为加入了这个音所以这个音阶听起来就会有一种西部牛仔，乡村的味道。

怎么按？

那在吉他上你可以怎么按呢？其实如果你把刚刚讲的理论搞懂了的话，你自己应该也可以找出来它们在指板上的位置了。就像我们刚刚讲的布鲁斯音阶就是一个五声音阶加入了 Blue Note。例如说我现在要找到 A 的小调布鲁斯音阶，我可以先把 A 的小调五声音阶音阶给找出来，其次加上能与根音形成三全音的音，这就是为什么你在练习一个音阶的时候你需要记住主音，这样你就可以快速的通过主音的位置找到其他音。 一般这个 Blue Note 会在下一根弦的下一品或者上一根弦的上一品。我知道这听起来有点复杂，让我们来举一个实际的例子好了。例如说我的主音是四弦七品的这个 A，那么它的 Blue Note 会在它的下一根弦的下一品，也就是三弦八品的 Eb，而另外一个 Blue Note 会在它的上一根弦的上面一品，也就是五弦六品。 这个方法在任何一根弦上都是行得通的，但除了二弦和三弦，因为吉他的调音系统导致所有其他相邻的弦都是完全四度，就二弦和三弦这两根弦之间的音程关系是大三度，所以说如果你的主音是三弦二品的这个 A，你要找下一根弦，也就是二弦上的 Blue Note 的时候你就需要多挪一个品格，但如果你是要找四弦上的 Blue Note 的时候你就按正常的方法推算就好了。 但如果你的主音是二弦十品的这个 A 然后你需要找上一根弦，也就是三弦上的 Blue Note 你就需要往上多挪一品，所以它就在三弦八品上，但是如果你要找的是一弦上的 Blue Note 你就只需要按照正常的方式来就好了。所以简单的总结一下，只有每当你的主音和 Blue Note 在二弦和三弦上的时候你才需要挪动两品，否则都是一品。用这个方法你就可以给你小调五声音阶加上 Blue Note 了。 然后如果你把其中的主音改成 C 的话你就直接获得了一个 C 的大调布鲁斯音阶，不需要再重新算 Blue Note 了。

怎么练？

在你平常练习的时候你除了手指机能训练或者肌肉记忆练习之外，也就是音阶上下行这些训练，还有几件事需要你特别注意。那就是你需要在做这些练习的时候留意并且清楚音阶里的主音主音在各个指型的位置以及所谓的 Blue Note 在各个指型的位置。 记住主音位置的原因有非常多，我会说如果你现在就把这个工作做好了的话，日后肯定会对你的帮助非常大，而且进步也会非常快，因为很多时候吉他手是靠着音阶的主音来快速视觉化音阶中剩下的音的，就像是今天讲到的 Blue Note，我们是通过主音的位置和一点的指板逻辑来找到的，而不是死记硬背整个指板音阶。

怎么用？

在了解了什么是布鲁斯音阶以及它在指板上的位置之后，你可以怎么使用它呢？就像所有的音阶一样，你可以用它来作曲，即兴给伴奏加花，你可以把这些不同的音阶想象成是不同口味的调味料，就像我们视频开头说的，你可以用这个音阶来制造一个西方牛仔的风格。因为视频时长的原因我打算把具体怎么用这个音阶来即兴留给下一期。 如果你是刚学完这个音阶的话，先不着急着找一个伴奏带然后在上面即兴，你可以试着用你的乐感或者直觉然后无节拍器无伴奏无和弦的造几个你认为很有蓝调风格的句子，就像这样。或者这样。 又或者这样。你可以用这个方法先试着熟悉一下这个音阶，造一些简短的乐句，而不是简单的音阶上下行，毕竟即兴说白了就是临时作曲，这种造句的练习会对你的即兴能力也会有很大的帮助的，
