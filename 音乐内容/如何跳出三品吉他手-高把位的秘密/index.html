如何跳出三品吉他手？高把位的秘密！

2025年8月26日 ； 阅读时间约 9 分钟

为什么要学习七和弦琶音？

在开始正式讲解之前我们应该先搞清楚一件很重要事情，那就是为什么我们要学习七和弦琶音呢？我能给出的第一个理由就是练习七和弦的过程中，你会渐渐地了解你的指板，你会了解指板上各个地方准确的音名，并且你也能迅速地反应出它和其他音的音程关系，这无论是对你的即兴还是演奏都会有很大的帮助。其次学会了七和弦琶音，你就有机会在即兴的时候让你的音与伴奏吻合。这时候你可能就会问了，在一首 C 大调的歌曲里我用 C 大调音阶即兴怎么会和伴奏不吻合呢？ 明明把那些音阶指型背下来就好了，为什么一定要学和弦琶音呢？其实这也是为了拓展你的思维能力，思考和弦的能力，而不只是音阶。这样的思维能够让你在即兴的时候弹出与当前和弦匹配的音，而不只是停留在整首曲子的调上，因为如果你的思考范围只局限于整首曲子的调上面的话，那么你还是有几率弹出一些不和谐的声音的，譬如说你有可能会在 G7 和弦上面反复强调 C。我们都知道如果 C 和 G7 一起弹会非常不和谐，因为 C 不是一个可用的和弦外音。就像这样。 尽管这些音都是来自 C 大调音阶，但他们听起来就是会不和谐，因此你需要了解这些琶音来做到与这首歌的和声进行听起来融洽。我们来听一下如果我们强调和弦内音来在一个 G7 上面即兴，效果会听起来怎么样。我们再听一遍一开始的例子。然后我们再对比一下这个用了和弦内音来即兴的例子。虽然第一个例子听起来不会特别难听，但是反复在 G7 上强调 C 会让整个旋律听起来是悬在空中的，缺少一个归属感，和谐感。 相比之下，在用了和弦内音来即兴，我们的旋律与伴奏之间就会更匹配。让我们再听一遍这两个例子。最后一个我能给出的学习和弦琶音的理由就是：这是一个任何乐手都需要知道的基本功，无论你是弹摇滚、古典、爵士、雷鬼还是流行等音乐风格。另外，学会琶音对你的创作能力以及即兴能力也会有很大的帮助。 虽然一提到即兴大家第一个联想到的就是爵士风格，但是无论你喜欢什么音乐风格，即兴是一个所有音乐人都需要掌握的技能，而琶音就能非常有效的提高你的音乐词汇量，来帮助你完成即兴。

如何按出七和弦琶音？

在吉他上算出七和弦琶音的方法有很多种，我个人认为最适合初学者的方法就是通过 CAGED 系统来算出七和弦琶音。CAGED 系统可以让我们找到五个不同的大和弦的按法，从中我们也可以算出五个七和弦的按法，因此我们也会有五个不同的大七和弦琶音。 它们分别长这样，这是 A 指型里的大七和弦琶音。这是 G 指型里的大七和弦琶音。 然后这个是 E 指型里的大七和弦琶音。D 指型。 最后是 C 指型。当然，中间这个 G 指型里的大七和弦只以琶音的形式存在着。 如果你要算出同根音的属七和弦，你只需要把其中的七音降低一个半音就好了，就像这样。同根音的小七和弦就只用把其中的三度音和七度音降低一个半音就好了，就像这样。 同根音的半减七和弦就要把三音、五音和七音都降低半音就行了，就像这样。

如何练习七和弦琶音？

知道了这些七和弦的按法之后，我们该如何练习呢。其实关于这个话题我很早之前就有做过一个相关视频，强烈推荐你在看完这期后去看看。在这里我想推荐两个基础的练习给刚入门的初学者。第一个就是把自己限制在一个大调音阶指型里，然后在这个指型里弹奏顺阶七和弦琶音，刚开始练习的话可以把 BPM 设置成 60 或者更低，就像这样。 第二个是纵向练习，我们先选定一个七和弦琶音指型，例如这里我选择 C 指型里的七和弦琶音，之后我就纵向的用这个指型弹奏顺阶七和弦琶音，和上个练习一样，在刚开始速度可以是 60 或者更低，等熟练了之后再慢慢提升，就像这样。当然了，在自己练习的时候千万不要忘了其他指型的七和弦琶音。那如果你觉得你能很好的掌握了这两个练习，并且能够迅速地在指板上找到任意七和弦琶音的时候，那么你就可以去看我之前做的这个视频里来探索其他更进阶的练习方式。

一句话先抓重点

这篇内容最重要的一点是：先听出功能，再做最小幅度替换和连接，你的和声就会既清晰又有色彩。

练习建议

建议你用“慢速短循环 + 录音回听”的方式练，每次只修一个变量。 先做单和弦或两和弦连接，再扩展到完整进行，确保每轮都能听见进步。
