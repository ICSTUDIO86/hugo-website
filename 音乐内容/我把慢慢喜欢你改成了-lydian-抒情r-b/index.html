我把慢慢喜欢你改成了 Lydian？！抒情R&B？

2025年2月13日 ； 阅读时间约 9 分钟

没错，今天就是要带你写出这样的和声效果，我会告诉你如何把一个正常大小调的歌曲改成 Lydian 调式，同时带你更深入的了解这些现代调式彼此之间的不同之处，在视频的最后我会告诉你一些编曲上的巧思，让你的曲子的整体和声更能够达成一致性，从而体现出 Lydian 调式的色彩。那我们开始今天的视频吧。

什么是 Lydian 调式？

虽然我说了让你先看完上期视频然后再回来观看现在这部，但我知道还是有很多人不会这样去做，所以我打算先给那些还不知道什么是 Lydian 调式的人做一个简单的讲解。所谓的 Lydian 调式是一个第四音升了半音的大调音阶。 如果想获得一个 C Lydian 音阶你就先列出一个 C 大调音阶，然后把其中第四个音，也就是这个 Fa，升高半音，这样你就得到了一个 C Lydian 音阶，而新获得的这个升 Fa，也就是升四音，就是这个音阶里的特色音。

Lydian 的和声进行

那么在一段和声进行里你可以怎么体现出 Lydian 调式独有的声音色彩呢？你需要强调两个元素：音阶的主和弦和带有特色音的和弦。在今天这首 C Lydian 的曲子里我要强调的主和弦可以是 ==C 和弦或者 C 大七又或者这个更具有 Lydian 色彩的 Cmaj7#11 和弦==。 接着我要强调的是带有特色音的和弦，在这个情况就是我们的==二级和弦，D，四级和弦，F#dim，然后七级和弦，Bm==。你只要不断重复地从一个特色和弦接到主和弦，你就可以制造出一个具有 ==Lydian 色彩的和声进行==。

用 Lydian 给旋律重配和声

知道了这些信息之后，给一段旋律配上 Lydian 和声对我们来说就很简单了。我们就只需要让曲子的和声在刚刚说的两种和弦之间来回切换就可以了。因为我们的目标是把这首歌改成 C Lydian，所以我们当然想要 C 和弦会出现在开头，结尾或者其他重拍上，在这之后我们要做的就是把特色和弦塞进歌曲里剩余的地方就行了。在这里我简单地讲一下换和弦的频率。你应该多久换一次和弦？又或者是什么时候才到换和弦的时机？ 其实在这个上面你是自由的，这更多取决于曲子的速度，风格和你想表达的内容。例如说在今天这个例子里，原曲的每小节里有两个和弦，但我因为个人的品味改成了一小节换一次和弦。尽管如此，在这个曲子里，还是有三个小节换和弦的次数超过了一次。 首先在第==六小节==里，我加入了一个 B 和弦，是因为这里有一个离调的音，所以我就加入了一个离调的和弦，为的是保持一个和谐感，但这个和弦也不会破坏我们想要的 Lydian 调式的色彩，是因为 B major 的和弦音也有我们需要的特色音，也就是 F#。再来是第八小节的 Bm 和 Cmaj7，这里我会想要两个和弦是因为这一段的乐句迎来了一个小停顿，然后我想让这最后一个音停在主和弦上，但我又不想连续两个小节的主和弦，所以就塞了一个 Bm。 最后这里是因为要迎来这个段落的句号了，所以放两个特色和弦来增强最后一个主和弦的听感，==让主和弦的稳定感更强一点==。现在我们来对比一下两个版本，一个是原版的《慢慢喜欢你》，一个是改编后的版本，为了让你更能听清楚和声上的变化，==所以待会的例子里只有主旋律和电钢琴。==

Pedal Note

如果你想再进一步地强调 C Lydian 色彩的话，你也可以放一个 Pedal Note，在中文里我们管它叫作持续音。顾名思义，持续音就是一个无论当前段落里的和声或者旋律怎么变化都保持一样 音高的音。一般在曲子里都是以低音的形式出现的。我们来听一下，如果我们把低音全部改成 C 听起来会怎么样吧。然后我们听一遍没有持续音的版本。然后再听一遍有持续音的版本。

编曲上的小巧思

最后是一点点的编曲上的小巧思。这编曲方面我没有选择让低音部分一直演奏 C，是因为我想让贝斯声部活跃一点，有一点和声变化，所以我让这个持续音以另外一个形式出现了。我用了一个合成器在中音区的位置演奏了一个 C 的持续音。这个持续音在曲子后半段还会高一个八度，来迎合曲子的变化。那最后让我们再听一遍改编后的曲子吧。
