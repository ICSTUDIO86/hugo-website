如何人性化练习音阶&琶音？吉他即兴入门

2024年12月23日 ； 阅读时间约 11 分钟

在之前这集为什么练习音阶这集我有说到，我们学习音阶是为了能在实战中用到而不是为了练而练。我举了像是即兴这样的例子，如果你能用一个音阶顺畅的即兴的话那就说明你大概已经掌握了这个音阶。那我们该怎么练习才能达到这样的效果呢？如果你有练过一段时间的音阶，你可能也发现了，那就是好像光练习音阶上下行，或者音阶模进并不能让我们完全的掌控音阶。 我们的确可以靠这些方式记住音阶的形状和它们在指板上的位置，并且有效的训练我们的手指机能，但是在实际情况我们还是没有办法在一个音乐上面即兴，还是不能完全随心所欲的使用所学过的音阶。那这究竟是为什么呢？这是因为我们在练习这些机械式的训练的时候通常会缺乏思考，而一味的追求速度，从而导致在实际情况的时候我们会随着肌肉记忆弹奏着那些机械式的旋律线，而不能弹奏一个有逻辑的旋律线。 要弹出一段漂亮的即兴演奏，除了掌握音阶指型之外，关键在于能够在正确的节奏上弹出正确的音符。而我们的问题就在于只掌握了音阶的形状，而缺少跟着一段音乐演奏的练习。

音阶

今天的目标就是要告诉你如何练习这些音阶，以便你可以更清楚的知道你演奏时弹奏那些音，从而让你也可以开始一些简单的即兴。我们今天的例子都会用 CAGED 系统里的 G 指型，但如果你自己练习的话也千万不要忘记其他的指型。### 琶音 就像我们一开始说的一样，如果你要即兴的话，光练习音阶上下行和类似的练习是不够的。因为在实际情况下，我们弹奏的东西并不会那么的规律，我们需要根据当下的和声来决定要弹奏的音，这样我们才会得到一个和谐的声响。 如果目前和弦是 C 和弦，我们在上面弹奏 C 和弦的和弦音，听起来就会和当下的和声很融洽。我们可以通过练习琶音来达到这个目的，如果我们了解了一个音阶指型里的琶音，我们就可以轻松地反应出当下需要的和弦音会出现在指板上的哪个地方。所以我觉得在我们正式熟悉音阶里的七个音之前，我觉得熟悉和弦琶音会是一个更重要的事情。我不建议你以音阶 顺阶的方式练习琶音，也就是说从 C 一路琶到 Bdim。 我建议你上网找一个伴奏带或者自己做一个然后跟着上面的和声进行练习琶音，这样的好处有：1. 你会更贴近实际情况，因为在你真正要即兴的时候，曲子的和声进行肯定不会是从 Cmaj7 到 Bø7，反而可能是这样 2. 因为有伴奏带，所以你即兴的时候会感觉到一种互动，从而感觉更有趣和更好玩 3. 如果你是自己做伴奏带的话，还可以提升你的编曲水平 4. 你还可以提升小节感，节奏感。 因为一首伴奏里面通常给你打拍子的不是简单且明了的节拍器，而是歌曲里面的节奏组。在刚开始跟着伴奏带练习琶音的时候，我建议你一拍一个音，然后每次到换和弦的时候你就从新和弦的跟音开始。之后，如果你觉得这个对你来说太简单了，你就可以在每次到新和弦的地方的时候不从跟音开始，而是离你目前最近的和弦音开始，然后从六弦一直琶到一弦，就像这样。当然，你也可以使用八分音符，三连音等节奏。 最后，如果你觉得你练的差不多的时候，你就可以开始用琶音来一些简单的即兴，就像这样。### 音阶 如果你已经对这首曲子的琶音非常熟悉了之后，然后你也想让你的即兴更加有趣一点的话，你就可以试着加入琶音以外的音了。练习的方式也是一样，我们还是跟着我们的伴奏带，但这次是一拍两个音，我们这次要做的是，把之前的琶音以外的音当作我们的经过音，然后我们每到一个新的和弦的时候就从当下的跟音开始搭建我们的音阶，就像这样。 你也同样可以在换和弦的时候从离你最近的那个音阶的音开始，但你要保证你的和弦音是在正拍上的，就像这样。

即兴

在你大概掌握以上内容的时候，你就可以试着在伴奏带上随便即兴几些简短的乐句。这里有一个非常好的练习就是你要限制你自己，让你自己弹的每一个乐句都要结束在一个和弦内音上面。例如说你可以让你的乐句只结束在和弦的三音上（audio 例子）。也可以让他结束在五音上或者主音上。注意：这是个会需要时间的练习，如果你是刚入门音阶的初学者的话，你应该会需要一段时间来掌握大调音阶里的顺阶七和弦或者三和弦琶音。 如果你一开始练习的话没有看到明显的进步，那都是正常的，千万不要因此而放弃。要能很好很自如的即兴并不是一件容易的事情，这些能力都是要慢慢积累的。所以一定要加油。虽然里面还是有一些比较机械式的练习，但区别在于，你是跟着伴奏练习的，而不是节拍器，而且一首伴奏里的和声进行也会更符合实际情况。怎么练习琶音？ 跟着和声进行，不开节拍器，琶音，从根音开始，之后上下行，如果一首歌只用琶音即兴的话就不会有什么错音的问题了；等你觉得你很熟悉了就可以打开节拍器 练哪些琶音？CAGED 里面的琶音 练哪些音阶？CAGED 怎么简单的记住它们 怎么练习音阶？

一句话先抓重点

这篇内容最重要的一点是：先限定可用材料，再把句子的起点、重音和落点安排清楚，即兴会立刻稳定很多。 先锁定主音与指型位置，再把听感和手型对应起来，音阶才会真正变成可用语言。

练习建议

建议你用“慢速短循环 + 录音回听”的方式练，每次只修一个变量。 先限制两到三个动机做问答，再逐步扩展到完整段落，避免一上来就跑满整条音阶。
