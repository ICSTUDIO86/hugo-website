厉害的人都是怎么用副属和弦？爵士且 R&B 的和声进行

2024年12月11日 ； 阅读时间约 8 分钟

核心内容

在开始讲解副属和弦之前，我们先来聊一下属和弦在一首音乐里到底有着一个什么样的作用？为什么它那么重要？如果你看过之前这期视频，你应该知道属和弦在一首曲子里通常会制造一个紧张感，并且有一个需要解决会主和弦的倾向。属和弦就像游戏中的“大 Boss”，玩家需要直面 Boss 并将其击败，然后解救公主，这过程就等同于作曲家解决属和弦制造的紧张感。不过正是因为有了属和弦制造的紧张感，所以主和弦的稳定感就会听起来更加的稳定。 我们来听一下以下这个例子（例子 C-G7-C）。你应该可以听出来，第二个 C 和弦会比第一个 C 和弦听起来更有满足感和解决感。但是如果我今天的目标和弦不再是 C 和弦了，而是 F 和弦。那我应该怎么让 F 和弦也有 C 和弦那样的满足感呢？我可以把 F 和弦当作临时的主和弦，然后在它前方放一个比 F 高完全五度的属和弦，也就是 C7 和弦。这样我就可以让 F 和弦听起来像 C 和弦一样有满足感了。我们来听听看。 像这种临时安插的属和弦在乐理上被称之为副属和弦。这个技巧在现代音乐里面非常常见，例如周杰伦的这首晴天。副属和弦的应用让这一句的前半部分充斥着不安和不和谐感，这也正对应了歌词”还要多久“这种等待的迫切和不安的感觉，但这种不安的感觉在后半句被解决了，就像是把前面那股憋着的能量释放出来了一样。这种解决感也正对应了歌词“在你身边“的稳定感。我们一起来听一下。我们从小就听过的生日快乐歌也有应用我们今天说的技巧。 待会听例子的时候注意一下这些属和弦所带来的一种蓝调，爵士且 R&B 的感觉，也注意这里 C7 所带来的离调感和紧张感以及后面的解决。那你可以怎么在你的曲子里使用副属和弦呢？其实这个问题没有准确的答案，这就和很多重配和声的过程一样，没有绝对的对与错。这都是取决于个人喜好和审美。不过我这里可以给你一些基础的使用方法。首先你可以用副属和弦丰富一段和声进行的情绪，让紧张的地方更紧张，让解决的地方更有解决感。例如说我有这样一个和声进行。 如果我想让这段和声进行听起来更有趣一点，我就可以把这里的 Cm7 改成 C7 和弦，这样就可以制造出明显的情绪变化。我还可以让副属和弦和一个离调和弦搭配起来一起使用，这也是一个让你的和声进行听起来很现代的技巧。在以下这个例子里，F 和弦并不属于 Ab 大调的，但是我还是我让 F 和弦出现在重拍并且当作一个目标和弦使用，然后在它前面放一个副属和弦。这样听起来就会有那种 R&B，蓝调或者爵士的感觉了。我们来听听看。 我们来听一遍没有用上以上技巧的版本。再听一遍改编后的版本。你也可以用一系列的副属和弦来制造一个特别的听感。例如说我有这样一个旋律。我可以把最后这个和弦当作目标和弦，然后前面放一系列的副属和弦。就像这样。最后我想说，虽然副属和弦是一个非常好用且好听的技巧，但是也要注意不要用太多，毕竟物以稀为贵，一旦听众习惯了这个听感，那么副属和弦所带来的的效果也会大大减少，而且还会让你的歌听起来十分的油腻。 不过还是那句话，一切以你的审美为主，如果你觉得好听，那么一切规则都是可以被打破的。

一句话先抓重点

这篇内容最重要的一点是：先听出功能，再做最小幅度替换和连接，你的和声就会既清晰又有色彩。

练习建议

建议你用“慢速短循环 + 录音回听”的方式练，每次只修一个变量。 先做单和弦或两和弦连接，再扩展到完整进行，确保每轮都能听见进步。
