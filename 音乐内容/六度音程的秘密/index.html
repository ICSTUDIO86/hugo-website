六度音程的秘密！

2025年4月17日 ； 阅读时间约 7 分钟

怎么按

在吉他上的六度音程一共有两种按法：第一种是在相邻的两根弦上的六度音程，如果拿 C 跟 A 这组大六度来举例的话，那它们在吉他上看起来就会像这样。但由于这个按法里的大六度需要很长的手指跨度，所以一般情况下更常会用到的是这个按法里的小六度，就像这样。 第二种按法相对来说会更常见，因为不需要非常大的手指跨度。这个按法里的六度音程会在相隔一根弦的两根弦上面。 还是拿 C 跟 A 举例。同样的 C 跟 A 在吉他上用第二种按法看起来就会像这样。 要获得一个小六度的话你只需要把最高音往上挪一品，降低一个半音就好了，就像这样。

怎么记

使用这些六度音程的难度其实并不在于记住这几个按法，而是需要你能够快速地反应出它们在音阶指型里的位置同时有办法跟着和声弹出和谐的六度音。因此你需要针对这方面做一些训练来在这些音阶指型里视觉化这些六度音程。你可以做的第一步就是在一个大调音阶指型里弹六度上下行，就像这样。或者横向的在吉他上弹六度上下行，就像这样。所以你要做的就是把六度音里最高的那个音当作参考音，这样的话，和声方面的运作逻辑就会和三度音程一样。 因为每当你的参考音是和弦内音的时候，那么它下方的六度音听起来也会与当下的和弦融洽。因为这两个音本质上还是三度音，只不过上面的那个音被移低了一个八度。知道了参考音之后，那我们就来到最后一个问题：我要怎么知道参考音的下面是一个大六度还是小六度？其实这很简单，如果这个参考音会与上方的三度形成一个大三度的话，那么它就会与下方六度形成一个小六度。相反，如果这个参考音会与上方的三度形成一个小三度的话，那么它就会与下方六度形成一个大六度。 举个例子，在 C 大调音阶里 E 会与上方三度音，也就是 G，形成一个小三度，所以他就会与下方六度音，也就是低一个八度的 G，形成一个大六度。音阶里的 F 会与上方三度音，也就是 A，形成一个大三度，所以他就会与下方六度音，也就是低一个八度的 A，形成一个小六度。

怎么弹

一句话先抓重点

这篇内容最重要的一点是：先听出功能，再做最小幅度替换和连接，你的和声就会既清晰又有色彩。

练习建议

建议你用“慢速短循环 + 录音回听”的方式练，每次只修一个变量。 先做单和弦或两和弦连接，再扩展到完整进行，确保每轮都能听见进步。
