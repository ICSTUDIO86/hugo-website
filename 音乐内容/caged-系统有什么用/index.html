CAGED 系统有什么用？

2026年1月2日 ； 阅读时间约 11 分钟

今天我就要用一支影片来带你认识什么是 CAGED 系统以及你到底可以拿它来干嘛。

什么是 CAGED 系统？

那到底什么是 CAGED 系统呢？CAGED 系统其实就是一个帮你在指板上“找位置”的系统。它并不是教你新的和弦，而是用五个你早就熟悉的开放和弦形状，去理解同一个和弦在不同把位上的分布。这五个形状分别是：C、A、G、E、D。比起把它们理解为 C 和弦，A 和弦，G 和弦之类的，你现在可以把它们理解为一个形状，一个指型，第一个 C 和弦你可以把它理解为 C 指型，第二个 A 和弦的话就把它理解成 A 和弦指型。 CAGED 系统正是通过这五种开放和弦形状，把同一个和弦在指板上的不同位置给连接起来，它们会跟着 C-A-G-E-D 这个顺序循环覆盖整个指板，从而形成一张可视觉化的指板地图。例如我们从 C 和弦的 C 指型开始，之后沿着指板往高把位移动，就会依次遇到 C 和弦 的 A 指型、C 和弦的 G 指型、C 和弦的 E 指型、C 和弦的 D 指型，而当继续向上延伸时，又会回到下一个八度的 C 和弦的 C 指型。 这个和第一个和弦用的是同一个指型。如果你觉得这个顺序太难记了，你可以暂时不记这个顺序，但你需要知道一件事：那就是同一个和弦，一定会在指板上以这五种形状出现。让我们用 D 和弦再试一遍：我们从 D 和弦的 D 指型开始，沿着指板向高把位移动，接下来会依次遇到 D 和弦的 C 指型、D 和弦的 A 指型、D 和弦的 G 指型、D 和弦的 E 指型，再继续向上，又会回到下一个八度的 D 和弦的 D 指型。 这说明无论从哪一个指型起步，C–A–G–E–D 都会以循环的方式覆盖整个指板，把同一个和弦首尾相连地展开。

CAGED 是其他和弦的基础

到这里为止，我们讨论的仍然是 CAGED 系统最基础的概念。但需要强调的是：这些看似基础的内容，并不是“只给初学者看的”。恰恰相反，它们是后续大量演奏技巧和和弦理解的出发点。接下来我要讲的部分，如果你是刚学吉他不久，你可能不需要一次听懂。但如果你已经开始接触横按、对和弦构造有一个基础的理解的话，那这可能会对你非常有用。这五个和弦的意义并不只是帮你在不同把位上找到同一个和弦。 更重要的是，它们为我们提供了一套理解和组织其他和弦指型的参考框架。简单来说，你可以用这五个指型推算出更多其他和弦指型。第一个获得新和弦的方法，就是通过将这五个指型沿着指板上下移动，变成完全不同的和弦。一个最典型的例子，就是许多初学者的“噩梦”——F 和弦。它本质上就是 E 和弦整体向上移动一品 得到的，这也正是我们所说的 F 和弦的 E 指型。而这还只是最基础的用法。 只要你对这五个指型中的音稍作修改，就可以得到更多不同类型的和弦。例如，如果你把 A 和弦中的三度音降低半音，你就得到了 Am 和弦；之后你再把这个 Am 指型整体向上移动两个品位，就会得到 Bm 和弦。再比如常见的 小横按 F 和弦，其实就是大横按 F 和弦 去掉了两个最低音而已。 就连很多七和弦的指型，也都可以追溯到这五个基础和弦：例如 Cmaj7 的 Drop 2 指型，你也可以感觉到 C 和弦的 A 指型的影子，只不过其中的一个根音降低了半音，变成了大七度。再例如，C7 和弦，则是从 C 和弦的 C 指型 出发，把原本的五度音改成了 ♭7 音。不过因为 C7 这个指型在沿着指板上下移动同时，因为手指数量的受限，我们会放弃一弦上面的音，选择把它给制住，只弹中间四个音。 像这种因为手指受限而放弃一些弦的列子也有很多，例如一些 add 2 和弦。add 2 和弦可以通过把和弦里的一个根音升高一个全音而获得。你还可以通过把和弦指型里的三音改成二音来获得一个 sus2 和弦，又或者把三音改成四音来获得一个 sus4 和弦。这样的例子数不胜数。不过当你理解了核心理念，就会发现：你平时弹到的绝大多数和弦，几乎都可以看到这五个基础开放和弦的影子。 只要稍微改动指型里的某个音，再将它们在指板上自由移动，你就能构建出几乎所有常见的和弦类型。这正是 CAGED 系统在实际演奏中真正强大的地方。现在让我们简单的总结一下关于 CAGED 系统的两个特征：1. 第一，CAGED 系统用 C、A、G、E、D 这五种开放和弦形状，帮助你理解同一个和弦如何在指板上不同的位置以不同指型出现，并按照 C–A–G–E–D 的顺序首尾相连、循环分布。 2. 第二，通过对这五个基础和弦指型进行移位、删减、替换音（例如改变三度音、加入七度音、修改低音等），你就可以构建出小和弦、七和弦、扩展和弦以及各种实用的演奏指型。

一句话先抓重点

练习建议
