刹车和弦，四级小和弦都来自于这个原理！让你的和声进行更加流行且不同。

2024年12月6日 ； 阅读时间约 7 分钟

核心内容

不知道你在听歌的时候会不会注意到这些声音。（后来的我们，我怀念的）。如果你们目前学琴有一段时间的话你一定有听过借用和弦或者调式互换这个观念，如果没有的话我这里大概简单的讲一下。借用和弦是指从其他同主音的音阶中临时借用的和弦。他们用于丰富和弦色彩和表达变化。我们来举个例子好了。例如说你要给一个 C 大调的曲子配和弦，我们最简单明了且直白的选择也就是 C 大调音阶里面的和弦。 但是作曲家为了不让曲子无聊，让曲子在和声方面有一点惊喜，作曲家通常会跳出 C 大调的范围，去别的调里借和弦。通常这个调会是向当前调的平行大小调。例如说 C 大调的曲子会向 C 小调借和弦。例如像孙燕姿这首《我怀念的》，这里的四级和弦就是从小调那里借过来的，这也是最常见的借用和弦之一，也就是四级小和弦。如果没有用的话听起来就没有那种伤心的感觉，我们来听听看。一般在大调流行音乐里最常用借用和弦基本都是来自于小调。 但是除了这些小调音阶你还可以用哪些其他音阶里借和弦呢？你还可以从同主音的其他 5 个调式里面借和弦。如果我们把我们可以借的和弦都列出来我们可以发现其实这有非常多的选择。只要旋律与和声之间不会打架的话的话，我们几乎可以把一个调内的任何和弦改成任意一个其他和弦。例如我有这样一个乐句，如果你不想从小调里面借和弦，你可以选择从 Phrygian 调式里面借。我们来试着把这里的 F 改成 Db。看看效果听起来怎么样吧。我们再来对比一下原版。 我们也可以用 C Lydian 调式里面 Bm 和弦来替换这里的 G7sus，听起来就会像这样。我们也一样对比一下原版。借用和弦的理论就是这么简单，你只要从当下音阶的主音开始列一个新的音阶，然后从里面选出一个于旋律搭配的和弦就好了。虽然原理简单，但是你该在曲子的什么地方使用它们呢？其实这个问题没有准确的答案，这就和很多重配和声的过程一样，没有绝对的对与错。这都是取决于个人喜好和审美，但是我这里可以给你一些简单且常用的使用方法。 首先最简单的使用借用和弦的方法就是把它当做两个调内和弦之间的一个经过和弦。例如说送别这首歌里，我们在这个主歌第一句的结尾加一个经过和弦，就像这样。我们先听一遍原版。再听一遍加上借用和弦的版本。你其实也可以把借用和弦当作目标和弦使用。由于听众原先期待的和弦被改动了，所以就会给听众一个惊喜感。我们拿生日歌来举例。我们把生日歌里面的这两句的结尾改成借用和弦来听听看效果怎么样。我们还是先听一遍原版，然后再听一遍改编后的版本。 记得把你的注意力放到这两句的结尾。我们很常提到的刹车和弦，升四半减七和弦，在 C 大调里就是 F#ø，这就是一个从 Lydian 调式里面借过来的和弦。所谓的刹车和弦就是一个用在一个乐句尾的和弦，通常会给人一种突然暂停，气氛或情绪被悬挂住了一样。莫文蔚这首《这世界那么多人》就是一个非常好的例子。调式 - 我们除了大小调还有什么？

一句话先抓重点

这篇内容最重要的一点是：先听出功能，再做最小幅度替换和连接，你的和声就会既清晰又有色彩。

练习建议

建议你用“慢速短循环 + 录音回听”的方式练，每次只修一个变量。 先做单和弦或两和弦连接，再扩展到完整进行，确保每轮都能听见进步。
